# Termux:X11 (Lenovo Xiaoxin Pad Pro GT / Yoga Pad Fork)

A [Termux](https://termux.com) X11 server add-on app.

## About this Fork
This is a personal fork created for my own use. It fixes touchpad gesture issues when using the **official Lenovo keyboard cover case**.

### Supported Device
* **Lenovo Xiaoxin Pad Pro GT (Model: TB710FU)**
* Also known as **Yoga Tab 11.1 AI / Yoga Pad** in some regions.

### Key Changes
* **Touchpad Fix**: Fixed tap-to-click and two-finger scrolling which were broken due to Samsung DeX detection logic.
* **Disclaimer**: This version has **NOT** been tested on any other devices or third-party keyboard cases. Use it at your own risk.
